# Cyrus

Zero-dependency AI memory system with hook-level rules enforcement for Claude Code.

> **Status:** Pre-alpha. Not yet published.

## What is Cyrus?

Cyrus gives your AI persistent memory across sessions — conversations, decisions, preferences — stored as plain markdown files. The differentiator: rules enforcement at the hook level that survives `--dangerously-skip-permissions`.
